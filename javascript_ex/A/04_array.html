      배열 : 비슷한 종류의 값을 하나의 변수에 할당

      새 배열 생성
      1. 빈 배열 + 값 추가
        var 배열명 = new Array(); // 빈 배열 생성
        var 배열명 = [];          // 빈 배열 생성

        배열에 값 추가
          배열명[인덱스 번호] = 값;

      2. 값을 생성하면서 배열 생성
        var 배열명 = [값, 값, 값, ...];

      3. 배열의 값 출력
        배열명[인덱스번호]

length

      배열의 length 길이, 배열 내 값의 (length)개수

join

      배열의 값을 하나의 문자열로 합침 (join)
      배열 -> 문자열 변경
      대상.join(); //기본값 콤마(,)로 연결해서 병합
      대상.join('-') // 하이픈(-)으로 연결해서 병합

reverse

      배열의 값의 순서를 역순으로 변경 - 원본을 수정
      대상.reverse();

sort

      배열의 값을 정렬 (오름차순, 내림차순)
      대상.sort();

      대상.push(새값);

shift

      배열의 첫번째 값을 제거

slice

      slice 자르다, 구간설정
      대상.slice(start, end);
      start 부터 end의 앞자리까지 복사

splice

      대상 특정 구간(값) 오려내기
      대상.splice(start, length);
      start부터 length 개수만큼 오려내기
      대상.splice(-1) 뒤부터 1개 오려내기
      대상.splice(-2) 뒤부터 2개 오려내기

concat

      문자열, 배열의 값을 병합 - 새 배열 생성
      A.concat(B)
      A배열과 B배열을 병합

Timer

setTimeout

      타이머 한번 세팅, 타이머가 다 되면 할일 실행

      문법 : 
      setTimeout(할일, 시간)
      
      할일 = 함수 = function(){실제 할일}
      시간 = ms 기준

      setTimeout(function(){}, 4000)

      타이머 끄기
      대상의 이름이 변수명으로 할당되어 있어야 함
      clearTimeout(대상);

setInterval

      일정 시간마다 할 일

      문법 : 
      setInterval(할일, 시간);

      타이머 끄기
      대상의 이름이 변수명으로 할당되어 있어야 함
      clearInterval(대상);

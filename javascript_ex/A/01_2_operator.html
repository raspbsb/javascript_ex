연산자

산술 연산자

      +, -, *, / 사칙연산
      % 나눈 나머지

문자열 더하기

    문자열 + 이어서 붙여줌 
    '문자' + 20 = 문자20
    '문자' + 20 + 30 = 문자2030
    20 + '문자' = 20문자
    20 + 30 + '문자' = 50문자
    '20' + 30 = 2030
    '20' - 30 = -10

증감 연산자

      i++ 출력 후 값을 1씩 증가
      i-- 출력 후 값을 1씩 차감

      후치 : 실행된 시점에서는 실행X, 실행된 이후에 적용하고 다음부터 적용

    
      ++i 값을 1씩 증가해 출력
      --i 값을 1씩 차감해 출력
      전치 : 실행된 시점부터 즉시 적용

복합대입연산자

      += 더한 결과
      -= 뺀 결과
      *= 곱한 결과
      /= 나눈 결과
      %= 나눈 나머지 결과

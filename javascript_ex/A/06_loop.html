      문법 : 
      for(초기값;조건;증감){
        반복할 구문
      }

while 반복문


      문법 : 
      초기값;
      while(조건){        //조건이 참일 동안 할 일
        반복할 코드
        증감
      }

do while 반복문


      일단 한번은 냅다 실행 (do 하고 while)
      그 이후로는 while과 동일

      문법 : 
      초기값;
      do{
        반복할 일;
        증감;
      } while (조건)

forEach 반복문

      배열의 각 원소를 반복
      대상 = 배열
      배열의 원소들 마다마다 할 일

      문법 : 
      대상.forEach(할일);
      할일 = 함수 = function(){}

      대상.forEach(function(){});
      대상.forEach(function(각원소, 인덱스, 배열 전체){});

for in 반복문

      
      객체 내의 속성의 값을 변수로 참조해서 반복
      대상 = 객체
      for(원소 in 대상) {
        반복할 일
      }
      
      문법 : 
      let st1 = {
        kor = 90,
        eng = 80,
        math = 100
      };

for of 반복문

      대상 = 배열
      문법 : 
      for(원소 of 대상) {
        반복할 일
      }

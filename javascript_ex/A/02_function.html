함수

        특정 작업을 수행하기 위해 코드를 묶어놓음
        필요할 때 재사용

        문법 : 함수 정의
            function 함수명(){수행할 작업}

        문법 : 함수 실행
            함수명();

함수의 구조

        함수 = 공장, 음식점
        함수 = 입력, 할일, 출력
        function add(a,b) {   // a,b 매개변수 parameter
            a,b로 할일 수행     // 할일
            return 결과        // 출력
        }
        add(10,20);           // 인수, argument

익명함수

      이름이 없는 함수
      function() {}

      익명함수는 반드시 변수에 할당을 해야 재사용 가능
      var add = function() {}

      익명함수 실행
      add();

      var add = function(a,b){}
      add(1,3);

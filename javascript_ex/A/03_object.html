Object

      객체 = 변수 + 함수를 합쳐서 재활용할 코드를 만듦
      object = property + method

      새 객체 생성
      1. 빈객체 + 값을 추가
      var 객체명 = new object(); // 빈객체 생성
      var 객체명 = {}; // 빈객체 생성

        값 추가
        객체명.property명 = 값;
      2. 값을 추가 객체 생성
      
      2. 값을 추가 + 객체 생성
      var 객체명 = {property명:value값, property명:value값, ...}
